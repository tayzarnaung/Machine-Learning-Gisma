Feature Engineering¶

I split features into categorical and numerical by converting categorical features into numbers and normalizing numerical features to help models perform better.

Random Forest (RF)¶

I deal with Hyperparameter Tuning to improve model accuracy by finding the best settings for RF.

In [73]:
# Define parameter grid for Random Forest
param_grid = {
    'n_estimators': [50, 100, 200],
    'max_depth': [None, 10, 20],
    'min_samples_split': [2, 5, 10],
    'min_samples_leaf': [1, 2, 4]
}

rf_model = RandomForestRegressor(random_state=42)

# Grid search with cross-validation (5-fold CV)
grid_search = GridSearchCV(estimator=rf_model, param_grid=param_grid, n_jobs=-1)

# Train the model with different hyperparameters
grid_search.fit(X_train_transformed, y_train)

# Get the best model with optimal parameters
best_rf = grid_search.best_estimator_
print("Best hyperparameters:", grid_search.best_params_)

Best hyperparameters: {'max_depth': 10, 'min_samples_leaf': 4, 'min_samples_split': 10, 'n_estimators': 200}

I evaluate the performance on Unseen Test Data before & after condition of tuning hyperparameter, which can be seen the best model with highest R2, lowest MAE, lowest RMSE
